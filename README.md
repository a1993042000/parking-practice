# 🅿️ 停車練習 · Parking Practice

俯視（上帝視角）的停車練習小遊戲。純 HTML5 Canvas + 原生 JavaScript，無需建置、無相依套件，可直接靜態部署。

## 功能
- **上帝視角**俯視畫面
- **可調車長 / 車寬**（滑桿即時調整，會影響迴轉半徑）
- **多種難停車位**：開放車位、垂直夾縫、路邊平行、窄車庫倒車、緊密平行
- **各種難度的迴轉 / 小巷**：S 型窄巷需精準操作
- 真實感的**單軌（bicycle model）車輛物理**：低速、會倒車、迴轉半徑隨打方向盤改變
- 碰撞偵測（SAT），撞牆會停下並計數
- 過關判定：車身需**完整進入車位**且**對正**、停穩

## 操作
| 操作 | 鍵盤 | 觸控 |
|---|---|---|
| 前進 | ↑ / W | ▲ |
| 倒車 | ↓ / S | ▼ |
| 左打方向盤 | ← / A | ◀ |
| 右打方向盤 | → / D | ▶ |
| 煞車 | 空白鍵 | — |
| 重置本關 | R | 重置鈕 |

## 關卡
1. 暖身：開進車位 ★
2. 垂直停車（夾縫）★★
3. 路邊平行停車 ★★★
4. 窄車庫倒車 ★★★
5. S 型窄巷 ★★★★
6. 緊密平行停車 ★★★★★

## 本機開啟
直接用瀏覽器打開 `index.html` 即可（或 `python3 -m http.server` 後開 localhost）。

## 部署
靜態網站，根目錄即為網站。已透過 **Vercel** 部署。
