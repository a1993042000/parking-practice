# 🅿️ 停車練習 · Parking Practice

俯視（上帝視角）的停車練習小遊戲。純 HTML5 Canvas + 原生 JavaScript，無需建置、無相依套件，可直接靜態部署。

## 功能
- **上帝視角**俯視畫面
- **可調車長 / 車寬 / 軸距**（滑桿即時調整，全部會影響操控與迴轉半徑）
- **真實前輪轉向物理**：運動模型參考點放在**後軸**，只有前輪會轉向、後輪固定，因此轉彎時後輪走比前輪更內側的弧線 —— 即真實的**內輪差 / off-tracking**，這正是停車最關鍵的手感
- **輪胎軌跡可視化**：橘色後輪、藍色前輪軌跡，直接看見內輪差（可關閉）
- **多種難停車位**：開放車位、垂直夾縫、路邊平行、窄車庫倒車、緊密平行
- **各種難度的迴轉 / 小巷**：S 型窄巷需精準操作
- 低速、可倒車，迴轉半徑 = 軸距 / tan(方向盤角度)
- 碰撞偵測（SAT），撞牆會停下並計數
- 過關判定：車身需**完整進入車位**且**對正**、停穩

## 物理模型
採用**後軸參考的單軌（kinematic bicycle）模型**：

```
後軸位置 (rx, ry)、車身朝向 θ、前輪轉角 δ
rx += v·cos(θ)·dt
ry += v·sin(θ)·dt
θ  += (v / 軸距) · tan(δ) · dt
前軸 = 後軸 + 軸距·(cosθ, sinθ)
```

因為積分參考點在後軸、且只有前輪有轉角 δ，後軸自然走出比前軸更小的轉彎半徑（後輪內切＝內輪差）。軸距越長、迴轉半徑越大、越難停。

## 操作
| 操作 | 鍵盤 | 觸控 |
|---|---|---|
| 前進 | ↑ / W | ▲ |
| 倒車 | ↓ / S | ▼ |
| 左打方向盤 | ← / A | ◀ |
| 右打方向盤 | → / D | ▶ |
| 煞車 | 空白鍵 | — |
| 重置本關 | R | 重置鈕 |
